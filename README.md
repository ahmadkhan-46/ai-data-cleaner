# 🤖 Smart Data Cleaner (AI-Based)

## 📌 Overview
This project is a Python-based data cleaning and preprocessing tool that transforms raw, unstructured datasets into clean and structured formats. It applies automated logic inspired by AI techniques to improve data quality and consistency.

---

## 🛠️ Technologies Used
- Python
- Pandas

---

## ⚡ Features
- Remove duplicate records
- Handle missing values
- Normalize text data (case formatting, trimming)
- Basic AI-inspired text cleaning logic
- Export clean dataset for further analysis
